# Website and Catalog Tasks

### Website / Catalog

PROVIDED:

- Claim-safe language controls.
- Course list and pending-approval status.

ACTION STEPS:

- Draft catalog copy that says the RCFE CETP approval packet is in preparation or pending, if public mention is necessary.
- Remove any implication that CDSS approval has been issued.
- Publish approved course numbers, vendor number, and certificate language only after CDSS issuance and Compliance clearance.

OUTPUT:

- Claim-safe RCFE CE website/catalog content.

DO NOT:

- Do not advertise approved status.
- Do not list CDSS course numbers or vendor numbers before issued.
- Do not imply administrators may use the courses for renewal credit before approval.
