# Post-Test and Answer Key - RCFE-CETP-009

| No. | Question | Answer key |
| --- | --- | --- |
| 1 | What makes a public RCFE CE claim unsafe before CDSS approval? | It may imply approval, course credit, vendor number, or course number that has not been issued. |
| 2 | Why should records be organized before CDSS submission? | Organized records reduce incomplete-response risk and support monitoring readiness. |
| 3 | What business-operation control supports audit readiness? | A document library with owner, status, source, and retention rule. |
| 4 | What should happen before using a CDSS vendor number publicly? | CDSS must issue the number and Compliance must clear the language. |
| 5 | Which Core category applies? | Business Operations. |
