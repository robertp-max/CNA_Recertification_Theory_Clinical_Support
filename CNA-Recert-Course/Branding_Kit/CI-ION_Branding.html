1. Visual Identity & Logos

This section defines the core visual representation of the CI Institute of Nursing. The primary logo should be used in light-themed areas, standard branding contexts, and application headers. The white logomark is strictly reserved for dark mode interfaces, solid color footers, and mobile application icons. Consistent clear space is mandatory.

White Logomark (Dark UI)

⚠

Usage Rules

  • Maintain padding around the logo equal to at least 20% of the logo's height.
  • Do not stretch or distort the logos. Always lock the aspect ratio when resizing.

4. UI Components & CSS Configuration

Standardized UI components ensure consistency across the application. Below are the primary button configurations, card layouts, and the required CSS variable root block for immediate integration into the project stylesheet.

Buttons

Primary: bg-[#FFC107], text-[#212529], border-radius: 6px.
Hover: Darken gold slightly.

Card Container

Cards are used to highlight course modules, announcements, and profile settings.

Background: #FFFFFF
Border Radius: 8px
Shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)
variables.css 
:root {
  /* Brand Colors */
  --ci-primary-maroon: #8B1515;
  --ci-accent-gold: #FFC107;
  
  /* Neutral/UI Colors */
  --ci-bg-surface: #F8F9FA;
  --ci-text-main: #212529;
  --ci-border-light: #DEE2E6;
  --ci-white: #FFFFFF;

  /* Typography */
  --font-heading: 'Poppins', 'Montserrat', sans-serif;
  --font-body: 'Open Sans', 'Inter', sans-serif;
  --font-mono: 'Roboto Mono', monospace;

  /* UI Styling */
  --radius-sm: 4px;
  --radius-md: 6px;
  --radius-lg: 8px;
  --shadow-card: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06);
}
