# 1. Executive Summary

CI Institute of Nursing is evaluating a separate optional California Board of Registered Nursing Continuing Education Provider approval lane for RN/care manager continuing education contact hours. This planning binder is not required for CNA recertification continuing education and is not part of the CDSS RCFE CETP application.

| Area | Status |
| --- | --- |
| Workstream | BRN Continuing Education Provider application planning only |
| Current status | Not Started / Optional Expansion |
| Approval posture | No BRN approval claimed; no RN continuing education contact hours claimed |
| Initial course lane | Care Manager / RN Case Management CE planning |
| Primary sources | BRN CEP application page, BRN application packet, BRN fee schedule, BRN processing times, Title 16 CCR Article 5 |

# 2. Why This Is Separate from CNA and RCFE

| Program lane | Governing approval path | Separation rule |
| --- | --- | --- |
| CNA CE | CDPH / TPRU | Keep CNA recertification materials outside this BRN folder. |
| RCFE administrator CE | CDSS / CETP | Keep RCFE LIC 9140 and LIC 9141 materials outside this BRN folder. |
| RN/care manager continuing education contact hours | California BRN CEP approval | Use this lane only if CI Institute of Nursing wants BRN-recognized RN continuing education contact hours. |

The BRN approves continuing education providers, not individual courses. BRN requirements must not be mixed into CNA or RCFE binders.

# 8. Care Manager / RN Case Management Course Fit Rules

Case management is potentially acceptable only when written as RN-practice-relevant content connected to direct or indirect patient/client care. Course fit remains [PENDING BRN CONFIRMATION] until the representative course is selected and mapped to CCR section 1456.

| Potentially acceptable themes | Do not include courses that are only |
| --- | --- |
| RN care coordination<br>indirect patient/client care<br>transitions of care<br>discharge planning<br>population health care coordination<br>chronic disease management coordination<br>medication reconciliation coordination from an RN role<br>legal/ethical issues in nursing case management<br>quality assurance and documentation<br>interdisciplinary communication<br>social determinants of health as they affect nursing care planning<br>patient education strategies<br>cultural and ethnic diversity in care coordination<br>burnout/stress in nursing practice, if framed within nursing practice | generic business development<br>financial investment<br>resume/job interview training<br>self-improvement<br>yoga/wellness not tied to patient/client care<br>lay-person training<br>non-nursing case management with no RN practice connection |

# 9. Advertising and Certificate Rules

| Control | Rule |
| --- | --- |
| Terminology | Use “contact hours,” not “CEUs.” |
| Provider statement | Include the BRN provider statement only after provider number is issued: “Provider approved by the California Board of Registered Nursing, Provider Number _____, for _____ contact hours.” |
| Pre-approval language | Before approval, use pending-approval language only and do not advertise approved status before issuance. |
| Certificate issuance | Do not distribute blank certificates; certificates must be completed by provider and issued within 90 days after course conclusion. |
| Certificate retention | Include the statement telling the RN to retain proof of attendance for 4 years. |
| Approval claims | Do not claim BRN approval, RN CE credit, care manager CE credit, or use any BRN CEP number unless issued by BRN. |
