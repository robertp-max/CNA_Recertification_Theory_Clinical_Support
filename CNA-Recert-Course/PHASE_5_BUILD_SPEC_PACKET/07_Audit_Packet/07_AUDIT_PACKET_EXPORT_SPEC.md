# Audit Packet Export Specification

**Purpose:** Define the learner-level evidence packet needed for compliance review, internal audit, and certificate support.  
**Retention recommendation:** Minimum four-year retention unless counsel sets longer retention.

---

## Audit Item Matrix

| Audit Item | Moodle Source | Export Method | Required? | Owner | Included in MVP? | Notes |
|---|---|---|---|---|---|---|
| Learner profile fields | User profile/custom profile fields | CSV/PDF profile export | Yes | Registrar | Yes | Legal name and CNA certificate number. |
| Identity confirmation | Acknowledgement quiz/form | CSV/PDF attempt export | Yes | Registrar | Yes | Include timestamp. |
| Online cap acknowledgement | Acknowledgement quiz/form | CSV/PDF attempt export | Yes | Compliance Reviewer | Yes | Confirms learner was warned about 24-hour online cap. |
| Theory completion report | Activity/course completion | CSV export | Yes | Moodle Admin | Yes | Required theory only; exclude optional clinical support. |
| Active-time report | Time plugin/custom report/admin review | CSV/PDF export | Yes | Moodle Admin | Yes | Include plugin settings or manual review note. |
| Interaction records | Lesson/H5P/Quiz attempts | CSV/PDF/screenshots as available | Yes | Instructional Designer | Yes | Must show learner input and feedback activity. |
| Final exam/test attempts and score | Moodle Quiz report/gradebook | CSV/PDF export | Yes | RN Educator / Registrar | Yes | Include pass/fail, score, timestamp, attempt count. |
| Signed statement/affidavit | Approved e-sign/wet-sign workflow | PDF/file export or form record | Yes | Registrar | Yes | E-sign method pending approval. |
| Certificate PDF | Certificate plugin/tool | PDF | Yes | Registrar | Yes | Only after gate pass. |
| Certificate issue log | Certificate plugin/tool logs | CSV/PDF export | Yes | Registrar | Yes | Include certificate ID/code and issue date. |
| Course version | Course settings/version log | PDF/CSV/change log | Yes | Moodle Admin | Yes | Freeze version before pilot. |
| Content version | Content inventory/change log | PDF/XLSX/CSV | Yes | Content Owner | Yes | Maps approved content to Moodle activities. |
| Certificate template version | Certificate template archive | PDF/screenshot/version log | Yes | Registrar | Yes | Include approved wording status. |
| Approval/NAC# metadata | Admin approval archive | PDF/scan/file reference | Yes | Compliance Lead | Yes | Certificate disabled if missing. |
| Manual override log if applicable | Restricted override log/form | CSV/PDF export | Conditional | Compliance Lead | Yes | Required when override exists. |
| Optional clinical support records | Clinical hub assignments/forms | Separate CSV/PDF folder | Optional | Clinical Coordinator | Yes if used | Label separately; not CE certificate gate. |
| PHI incident log if applicable | Restricted privacy incident log | Restricted export | Conditional | Privacy Officer | Yes if incident occurs | Do not include PHI in general packet. |
| Support tickets relevant to certificate | Help form/messages | PDF/CSV as available | Conditional | Support Lead | Yes if issue affects certificate | Include only relevant records. |

## Naming Convention

Use:

`AUDIT_PACKET_[LASTNAME]_[FIRSTNAME]_[COURSESHORTNAME]_[YYYYMMDD]`

Use certificate ID inside the packet index, not in every filename, to reduce accidental exposure.

## Packet Index Template

```markdown
# Audit Packet Index

Learner:
Course:
Completion date:
Certificate ID:
Packet generated by:
Packet generated date:

## Included Evidence
- Learner profile fields
- Identity confirmation
- Online cap acknowledgement
- Theory completion report
- Active-time report
- Interaction records
- Final exam/test record
- Signed statement/affidavit
- Certificate PDF
- Certificate issue log
- Course/content/certificate template version
- Approval/NAC# metadata
- Exceptions, if applicable

## Notes
Optional clinical support records, if included, are separate support documentation and are not online CE certificate gates unless written CDPH approval exists.
```

## Retention Note

Retain certificate-supporting evidence, completion records, exam/test evidence, signed statement/affidavit, approval records, certificate issue logs, and active-time reports for at least four years unless legal/compliance sets a longer schedule. If PHI is accidentally submitted, restrict incident records and follow privacy counsel guidance.
