# Moodle Mapping Notes

| Sandbox element | Intended Moodle tool/process | Notes |
|---|---|---|
| Module 0 orientation | Page or Book | Required activity completion by view. |
| Online cap acknowledgement | Quiz, Feedback, Choice, Assignment, or Checklist | Must acknowledge 48-hour renewal rule, annual minimum, and 24-hour online cap. |
| Legal name status | User profile field | Required certificate identity field. |
| CNA certificate number status | User profile field | Required certificate identity field. |
| Required theory activity | Page, Book, Lesson, SCORM/H5P placeholder, Assignment, or Quiz placeholder | Placeholder only in Prototype Build 1. |
| Required interaction/check | Forum, Assignment, Quiz question, Lesson question, Feedback, H5P, or equivalent | Must create auditable attempt or completion evidence. |
| Active-time status/manual hold | Candidate active-time plugin, grade item, checklist item, manual completion, or admin review hold | Active-time remains unvalidated until staging tests pass. Moodle logs alone are not enough. |
| Final exam/test placeholder | Final Quiz or test activity | Passing grade required before certificate availability. |
| Final signed statement/affidavit | Feedback, Assignment, Quiz acknowledgement, Checklist, or approved e-signature-capable process | Placeholder only until approved method is selected. |
| Certificate status | Certificate or Custom Certificate activity if available in staging | Staging-only or disabled until gate testing; no clinical hours. |
| Admin hold | Grade item, group, manual completion, restricted field, or admin-only checklist | Must block certificate while active. |
| Optional Clinical Support Hub | Page, Book, Label, section, or separate course shell | Optional, non-credit, non-gating. |
| Optional confidence check | Ungraded Quiz, Feedback, Choice, or H5P | Optional and excluded from certificate gate. |
| Optional documentation support | Page, Folder, Assignment placeholder, or Label | Non-PHI warning required; not certificate-gated. |
