# RCFE Reviewer Access Documentation

Delivery posture: all first-wave RCFE courses are documented as live-stream.

| Required reviewer-access element | Prepared documentation |
| --- | --- |
| Reviewer URL field | Reviewer URL - final verification required |
| Reviewer login/password field | Reviewer credentials - final verification required |
| Platform field | Live-stream platform - final verification required |
| Active participation method | Live attendance checks, polls/chat prompts, verbal check-ins, facilitator monitoring, post-test completion |
| Assessment/completion gate | Post-test plus attendance/participation verification before certificate release |
| Certificate release control | No certificate until attendance, participation, and assessment requirements are satisfied |
| Record-retention reference | Recordkeeping statements per course packet |
