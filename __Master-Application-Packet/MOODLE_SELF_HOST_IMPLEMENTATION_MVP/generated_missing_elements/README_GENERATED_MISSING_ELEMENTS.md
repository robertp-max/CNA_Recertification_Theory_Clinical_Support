# README - Generated Missing Elements

Status: Draft / Pending Approval. Every file in this folder is a "Generated Moodle MVP implementation artifact." Each was created because the controlling application packet lacked a directly Moodle-buildable artifact for that element. All content is conservative and SME-reviewable.

## Why these exist
The master application packet defines course structures, hours, and posture but does not provide ready-to-build Moodle artifacts for objectives-per-activity, attestation copy, remediation rules, a reviewer script, learner no-PHI warnings, or certificate gate copy. These files supply build-ready drafts so a Moodle administrator can construct the MVP without guessing.

## Review flags
- Content judgment items: SME review required before production use.
- Legal/compliance copy (attestations, certificate gate, no-PHI): legal/compliance review required.
- None of these are regulator-issued facts; no approval numbers, signatures, payments, or real credentials are invented.

## Files
- GENERATED_CNA_UNIT_OBJECTIVES_AND_ACTIVITIES.md - objectives + activity set per CNA unit (U01-U12) + Final.
- GENERATED_RCFE_COURSE_OBJECTIVES_AND_ACTIVITIES.md - objectives + activity set per RCFE course (001-009).
- GENERATED_BRN_MODULE_OBJECTIVES_AND_ACTIVITIES.md - objectives + activity set per BRN module (M01-M10) + Capstone.
- GENERATED_ATTESTATION_TEXTS.md - learner attestation/identity/no-PHI/completion attestation copy.
- GENERATED_REMEDIATION_RULES.md - failed-attempt remediation policy.
- GENERATED_SURVEY_REVIEWER_SCRIPT.md - reviewer-facing orientation/script.
- GENERATED_NO_PHI_LEARNER_WARNINGS.md - learner-facing no-PHI warning copy.
- GENERATED_CERTIFICATE_GATE_COPY.md - certificate gate/hold copy (pre- and post-approval).
