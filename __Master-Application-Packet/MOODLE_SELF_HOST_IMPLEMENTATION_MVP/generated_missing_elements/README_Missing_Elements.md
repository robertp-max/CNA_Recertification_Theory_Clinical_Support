# README - Moodle Missing Elements Draft Assets

Status: Draft / Pending Approval. These files are Moodle MVP implementation support artifacts. They are kept for source control and build review, but they are not included in the Google Drive end-user packet.

## Files
- `CNA_Unit_Objectives_and_Activities.md` - objectives and activity set per CNA unit (U01-U12) and final review.
- `RCFE_Course_Objectives_and_Activities.md` - objectives and activity set per RCFE course (001-009).
- `BRN_Module_Objectives_and_Activities.md` - objectives and activity set per BRN module (M01-M10) and capstone.
- `Attestation_Texts.md` - learner attestation, identity, no-PHI, and completion attestation copy.
- `Remediation_Rules.md` - failed-attempt remediation policy.
- `Survey_Reviewer_Script.md` - reviewer-facing orientation/script.
- `No_PHI_Learner_Warnings.md` - learner-facing no-PHI warning copy.
- `Certificate_Gate_Copy.md` - certificate gate/hold copy before and after approval.

## Review flags
- SME review is required before production use.
- Legal/compliance review is required for attestation, certificate gate, and no-PHI copy.
- No approval numbers, signatures, payments, or real reviewer credentials are invented.
