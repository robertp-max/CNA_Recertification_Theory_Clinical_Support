# CIION - Website Update Requirements

Status: Draft / Pending Approval

Guardrail: No public approval claim, enrollment for approved credit, or certificate release is authorized until the applicable agency approval metadata is issued and verified.

## Purpose

This source Markdown supports the public PDF `CIION - Website Update Requirements.pdf`. It defines the approval-safe continuing education website layer for the CI Institute of Nursing CE application tracks.

## Website Structure

The website must include a top-level Continuing Education section with these pages:

- Continuing Education landing page.
- CNA Renewal Online CE.
- RCFE Administrator CE.
- RN Case Management CE.
- Reviewer / Agency Access.
- Continuing Education FAQs.

## Pending-Approval CTA Rules

| CTA Category | Allowed While Pending | Prohibited While Pending |
| --- | --- | --- |
| General public | Join Interest List; Request Information | Enroll Now; Start Course; Earn CEUs; Get Certificate; Certificate Available Now; Approved CE |
| Reviewer or agency | Reviewer Access | Learner enrollment language or approved-credit marketing |
| Program pages | Request Information; Join Interest List | Any call to action implying current approval, active enrollment for credit, or certificate availability |

## Program Page Requirements

### CNA Renewal Online CE Page

- Display Draft / Pending CDPH Approval.
- State 24 online CE hours.
- State online CE only.
- Do not describe clinical hours.
- Do not claim full 48-hour CNA renewal completion.
- Use interest-list or request-information routing only while pending.

### RCFE Administrator CE Page

- Display Draft / Pending CDSS/ACS Approval.
- State 9 courses / 27 hours.
- Exclude the future 40-hour concept from the current public offering.
- Use interest-list or request-information routing only while pending.

### RN Case Management CE Page

- Display Draft / Pending BRN CEP Approval.
- State 30 contact hours.
- Use contact hours, not CEUs, except when explaining prohibited wording.
- Do not claim certification, credentialing, licensure expansion, or scope expansion.
- Use interest-list or request-information routing only while pending.

## Reviewer / Agency Access Page Requirements

- Keep reviewer access private, controlled, or unlisted unless the owner approves a public route.
- State that reviewer access is for signer, reviewer, or agency review only.
- Do not describe reviewer access as learner enrollment.
- Do not advertise reviewer access as approved-credit availability.
- Provide access-request routing and support contact handling.
- Keep no-PHI warnings visible for uploaded materials, support messages, and review comments.

## FAQ Requirements

| Question | Approval-Safe Answer Requirement |
| --- | --- |
| Is this approved yet? | Use Draft / Pending Approval until agency metadata is issued. |
| Can I enroll now? | Use Join Interest List or Request Information while pending. |
| Can I get a certificate now? | No. Certificate release is blocked while approval metadata and completion evidence gates are pending. |
| Does CNA include clinical hours? | No. The current CNA track is online CE only. |
| Does CNA complete all 48 renewal hours? | No. Describe 24 online CE hours only. |
| Does RCFE complete 40 hours? | No. The current RCFE package is 9 courses / 27 hours. |
| Is BRN measured in CEUs? | No. BRN language must use contact hours, not CEUs. |
| Who can access reviewer materials? | Authorized signer, reviewer, agency, or owner-approved reviewer accounts only. |
