## Module Description
Prepares the CNA to use objective and subjective observation skills and to report and record observations on appropriate documents using medical terms and abbreviations, within scope and with strict privacy protection.

## Measurable Learning Objectives
The learner will be able to:
1. Define key documentation terminology (objective, subjective, observation, assessment, MDS, Kardex, incident report, ADL).
2. Identify common medical word elements (prefix, root word, suffix).
3. Identify common medical abbreviations used in facilities (and respect facility "do-not-use" lists).
4. Define observation and list the senses used to observe a resident.
5. Distinguish objective from subjective observations.
6. Match charting document types to their use; describe accurate, timely, factual, scope-safe recording procedures and corrections; identify when/how to complete an incident report objectively.

## Required Learning Content Sections
- Medical word elements; common terminology and abbreviations.
- Observation: senses used; objective vs subjective.
- Charting document types (Kardex, MDS/RAI, care plan, incident report, flow sheets).
- ADL observation input for MDS.
- Recording procedures, timeliness, factual entries, corrections.

## Fictional / De-identified Resident Scenario (S-M15)
A fictional charting entry reads: *"Resident seems depressed and probably has the flu."* **Decision points:** (1) What is wrong with this entry (opinion + diagnosis)? (2) Rewrite it as an objective, scope-safe entry. (3) What must be reported to the nurse? **Scope guardrail:** CNAs document observations, not diagnoses or opinions.

## Learner Activities

## Documentation / Reporting / Decision Exercise
Convert subjective notes into an **objective entry** and complete a **simulated incident report** (fictional) using factual observations only.

## Knowledge Check Blueprint
8–10 items: terminology/word elements, abbreviations, objective vs subjective, charting rules, incident reporting.

## Assessment Items (seed)
1. (MCQ) Which is an OBJECTIVE observation? A) "Resident seems sad" B) "Resident ate 50% of lunch" C) "Resident is depressed" D) "Resident is fine"
2. (TF) A CNA may chart "resident has pneumonia" based on a cough.
3. (MCQ) When should a CNA chart care provided? A) At end of shift from memory B) Promptly and factually after the care C) Only if asked D) Next day
4. (MCQ) An incident report should contain: A) Opinions about blame B) Objective facts of what was observed C) A diagnosis D) Other residents' names

## Internal Answer Key Appendix (admin/faculty only — do NOT publish)
1 = B. 2 = False (observations, not diagnoses). 3 = B. 4 = B.
