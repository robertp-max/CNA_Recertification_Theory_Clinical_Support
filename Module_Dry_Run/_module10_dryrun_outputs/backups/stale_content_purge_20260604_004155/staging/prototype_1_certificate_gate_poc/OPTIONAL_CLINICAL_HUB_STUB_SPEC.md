# Optional Clinical Hub Stub Specification

## Purpose

This is the first optional clinical support stub for Prototype Build 1. It is not final content and does not create clinical-hour credit. It proves separation from the online CE certificate gate.

Use only these concepts for learner-facing labels: optional clinical support, skills refresh, practice support, documentation support.

Do not use "required clinical hours" unless explicitly saying this is not that.

### Draft Learner Copy

Optional Clinical Support Hub

Skills refresh, practice support, and documentation support.

This hub is here to help you prepare, organize practice support, and keep non-PHI documentation in one place. It is separate from your online CE certificate requirements unless CI Institute of Nursing gives you written CDPH-approved instructions.

This optional clinical support is not California CNA renewal clinical-hour credit by default. It does not replace CDPH renewal documentation unless CDPH approves that use in writing. Skipping optional clinical support does not block your online CE certificate if you complete all required online CE certificate steps.
