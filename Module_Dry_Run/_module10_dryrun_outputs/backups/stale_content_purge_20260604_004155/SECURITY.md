# Security

This repository must not store sensitive Moodle, learner, certificate, or production audit materials.

Do not commit:

- PHI.
- Learner records.
- Certificate PDFs.
- Production audit packets.
- Credentials, tokens, API keys, or `.env` files.
- Moodle backups.
- Private approval documents unless the repository is private and access-controlled.

Use synthetic staging data for Prototype Build 1. Store screenshots, exports, learner-data samples, audit evidence, and production evidence outside the repository unless explicitly approved for a private, access-controlled repo.
