# Standalone MVP Scope

This sandbox is a local proof-of-concept for Prototype Build 1 while Moodle access is blocked.

This standalone sandbox proves logic and flow only; it does not prove Moodle configuration, Moodle compliance execution, active-time plugin behavior, audit export behavior, role permissions, or live certificate issuance.

## In Scope

- Synthetic learner switching.
- Required gate status display.
- Deterministic certificate gate evaluation.
- Required vs optional separation.
- Optional clinical support non-gating proof.
- QA negative scenario display.
- Mock audit packet checklist.
- Non-production warnings.

## Out Of Scope

- Moodle configuration.
- Moodle plugin validation.
- Real learner enrollment.
- Real certificate issuance.
- Real active-time evidence.
- Real audit exports.
- Real screenshots or evidence files.
- Course content generation.
- PHI collection.

## Status

Ready for local review only. Moodle staging remains required before Prototype Build 1 can be accepted.
