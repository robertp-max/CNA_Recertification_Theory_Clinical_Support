# Limitations And Non-Production Warnings

This sandbox is not Moodle and is not production.

## Hard Limits

- No real certificate issuance.
- No live certificate enabling.
- No validated active-time plugin.
- No Moodle logs.
- No Moodle roles.
- No Moodle report exports.
- No production data.
- No real learner use.
- No PHI.
- No screenshots, audit evidence, Moodle backups, or private approvals.
- No credentials, passwords, or tokens.
- No course content generation.

## Required Follow-Up

Moodle staging access, admin-capable role confirmation, selected certificate tooling, selected active-time tooling or manual fallback, and evidence storage locations must be provided outside Git before Moodle staging configuration resumes.
