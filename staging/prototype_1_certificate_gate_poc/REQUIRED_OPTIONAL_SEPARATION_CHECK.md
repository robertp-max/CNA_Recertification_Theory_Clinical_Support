# Required / Optional Separation Check

## Purpose

This checklist proves required online CE progress and optional clinical support are visually and technically separate. Optional clinical support must not inflate required online CE progress and must not block certificate release.

| Check | Expected Result | Evidence | Pass/Fail |
|---|---|---|---|
| Required online CE section is labeled separately | Learner sees required online CE certificate requirements as separate from optional clinical support | Course screenshot | TBD |
| Optional clinical support landing page is labeled optional | Label uses optional clinical support, skills refresh, practice support, or documentation support | Course screenshot | TBD |
| Optional clinical support skipped does not block certificate | Learner completes required theory gates, skips optional support, and staging certificate becomes available if all required gates pass | Certificate status screenshot; optional completion report | TBD |
| Optional confidence check skipped does not block certificate | Learner skips optional confidence check and certificate availability is unaffected | Completion settings screenshot; certificate status | TBD |
| Optional documentation upload skipped does not block certificate | Learner skips documentation support/upload and certificate availability is unaffected | Upload settings screenshot; certificate status | TBD |
| Optional clinical support does not inflate required CE progress | Required progress remains based only on required online CE gates | Progress block/settings screenshot | TBD |
| Optional clinical support is not displayed as required renewal clinical hours | Learner-facing labels do not describe optional support as required clinical hours | Landing page screenshot; copy review | TBD |
| Optional clinical support is not graded | Optional confidence/support activities have no grade-to-pass and no required gradebook dependency | Gradebook/settings screenshot | TBD |
| Optional clinical support is absent from certificate restrictions | Certificate Restrict Access conditions do not reference optional support activities | Certificate restriction screenshot | TBD |
| Optional clinical support records are separately labeled in audit dry run | Optional status appears in a separate optional/support export, not required CE evidence | Audit dry-run export list | TBD |
| Simulation/virtual/LTI support is absent or labeled as not clinical hours | Prototype does not add simulation; any future placeholder says not clinical-hour credit | Course screenshot or absence noted | TBD |
| PHI collection is not present | Optional documentation stub warns against PHI and does not require PHI upload | Stub screenshot; privacy review note | TBD |

## Required Evidence Package

Store evidence with the QA cycle:

- Course page screenshot showing separated sections.
- Completion settings screenshots.
- Certificate Restrict Access screenshot.
- Required progress screenshot if a progress tool is used.
- Optional clinical support skipped test result.
